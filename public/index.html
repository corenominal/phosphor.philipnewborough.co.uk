PHOSPHOR // OVERVIEW

Phosphor generates strong, random passwords using the movements of your mouse or finger — not a predictable algorithm. The more you move, the more randomness is collected.

  1. COLLECT ENTROPY — Move your pointer around the glowing field. The ENTROPY bar shows how much has been gathered.
  2. GENERATE YOUR PASSWORD — Once the bar is full, click GENERATE PASSWORD to create a password from your collected entropy.
  3. COPY AND CLEAR — Use the COPY button, then RESET to wipe all data from memory.

// Your password is created entirely inside your browser. Nothing is stored on a server or sent anywhere.

TECHNICAL DETAIL
  • Your pointer coordinates (x, y) and timestamps are sampled continuously into an entropy pool (Uint8Array).
  • The pool is digested using SHA-256 via the browser’s built-in window.crypto.subtle API.
  • The resulting hash bytes are mapped to your chosen character classes using modular arithmetic to construct the final password.
  • All operations are performed locally. No external libraries or network requests are used at any point.
